◆ stAddListItem()

int stAddListItem ( stList *  list,
void *  value 
)

Removes a previously inserted node

Parameters
listThe list from which the node is to be removed.
valueThe address of the data you want to store. The data from the address will be copied instead of storing the pointer. That is, the data pointed by the pointer will be duplicated and stored. Any changes to original after storing will not affect the stored value. The values will be appended to end of the list.
Returns
: An integer signally success or failure of operation. (0) -> Successful (-1) -> Failed
